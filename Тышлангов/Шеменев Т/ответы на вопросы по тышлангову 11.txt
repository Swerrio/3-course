**1. Команды для работы с каталогами в Linux:**

**Навигация:**
- `pwd` - показать текущий каталог
- `cd [путь]` - сменить каталог
- `cd ~` или `cd` - перейти в домашний каталог
- `cd ..` - перейти на уровень выше
- `cd -` - вернуться в предыдущий каталог

**Создание и удаление:**
- `mkdir [имя]` - создать каталог
- `mkdir -p [путь]` - создать цепочку каталогов с родительскими
- `rmdir [имя]` - удалить пустой каталог
- `rm -r [имя]` - удалить каталог рекурсивно с содержимым
- `rm -rf [имя]` - принудительное удаление без подтверждения

**Просмотр и информация:**
- `ls` - список файлов и каталогов
- `ls -l` - подробный список с правами и владельцем
- `ls -a` - показать скрытые файлы (начинающиеся с .)
- `tree` - показать дерево каталогов
- `du -sh [каталог]` - размер каталога

**Копирование и перемещение:**
- `cp -r [источник] [цель]` - копировать каталог рекурсивно
- `mv [источник] [цель]` - переместить/переименовать каталог

**2. Порядок действий при завершении процесса в Linux:**

**1. Поиск процесса:**
```bash
ps aux | grep [имя_процесса]
```
или
```bash
pgrep [имя_процесса]
```

**2. Завершение процесса:**

**Корректное завершение (SIGTERM):**
```bash
kill [PID]
```
или
```bash
killall [имя_процесса]
```

**Принудительное завершение (SIGKILL):**
```bash
kill -9 [PID]
```
или
```bash
pkill -9 [имя_процесса]
```

**3. Проверка результата:**
```bash
ps aux | grep [PID]
```
или
```bash
pgrep [имя_процесса]
```

**Альтернативные утилиты:**
- `top` или `htop` - интерактивный мониторинг и завершение (клавиша 'k')
- `systemctl stop [сервис]` - для системных служб

**Важно:** Сначала использовать обычный `kill`, и только если не помогает - `kill -9`.
