**1. Реестр Windows**  
Это иерархическая база данных, содержащая настройки и параметры операционной системы, установленных программ и оборудования. Представляет собой двоичные файлы, которые загружаются в память при запуске Windows.

**Структура реестра:**  
- **HKEY_CLASSES_ROOT** - ассоциации файлов и COM-объектов  
- **HKEY_CURRENT_USER** - настройки текущего пользователя  
- **HKEY_LOCAL_MACHINE** - системные настройки для всех пользователей  
- **HKEY_USERS** - настройки всех загруженных профилей  
- **HKEY_CURRENT_CONFIG** - конфигурация оборудования при запуске

---

**2. Системный монитор и монитор ресурсов: отличие**  

**Системный монитор (perfmon.exe):**  
- Сбор и анализ данных производительности over time  
- Настройка счетчиков (CPU, память, диск, сеть)  
- Создание журналов и отчетов  
- Установка базовых показателей и предупреждений  

**Монитор ресурсов (resmon.exe):**  
- Мониторинг в реальном времени  
- Визуальное отображение использования ресурсов  
- Детальная информация по процессам, службам, сетевым подключениям  
- Возможность завершения процессов и анализа проблем  

**Ключевое отличие:** System Monitor - для долгосрочного анализа, Resource Monitor - для мгновенной диагностики.

---

**3. Назначение утилиты очистки диска**  
Удаление временных и ненужных файлов для освобождения места:  
- Временные файлы системы и приложений  
- Файлы корзины  
- Эскизы, кэш браузера  
- Очистка обновлений Windows  
- Очистка системных файлов (включая предыдущие установки Windows)  
- Файлы дампа памяти и ошибок  

---

**4. Дефрагментация жесткого диска: назначение**  
Оптимизация расположения фрагментированных файлов на HDD для:  
- Ускорения доступа к данным  
- Уменьшения времени поиска головками диска  
- Повышения общей производительности системы  

**Важно:** Для SSD дефрагментация не применяется - используется оптимизация (команда TRIM).

---

**5. Восстановление системы**  

**Назначение:**  
Откат системных файлов, реестра и настроек к предыдущему рабочему состоянию без потери пользовательских файлов.  

**Как проводить:**  
1. **Через Панель управления:**  
   Восстановление → Запуск восстановления системы  

2. **Через дополнительные параметры:**  
   - Удерживать Shift + Перезагрузка  
   - Поиск и устранение неисправностей → Дополнительные параметры → Восстановление системы  

3. **При невозможности загрузки:**  
   - Использовать установочный носитель Windows  
   - Выбрать "Восстановление системы"  

**Создание точки восстановления:**  
Панель управления → Система → Защита системы → Создать
